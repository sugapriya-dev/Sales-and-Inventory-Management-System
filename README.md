# 🏪 Supermarket Sales & Inventory Management System

A complete web-based system for managing supermarket operations including product management, sales, purchases, inventory tracking, and financial reporting.

## ✨ Features

- **Dashboard** - Real-time analytics and charts
- **Product Management** - Add, edit, delete products
- **Category Management** - Organize products by categories  
- **Supplier Management** - Track supplier information
- **Customer Management** - Maintain customer database
- **Sales System** - Process sales with payment tracking
- **Purchase System** - Manage purchase orders
- **Stock Ledger** - Track inventory movement
- **Financial Ledgers** - Bank, cash, supplier, customer ledgers
- **GST Management** - Handle GST slabs and calculations
- **Print Bills** - Generate printable invoices

## 🛠️ Tech Stack

- **PHP 8.2** - Backend logic
- **MySQL 8.0** - Database
- **JavaScript & AJAX** - Dynamic interactions  
- **HTML5/CSS3** - Frontend interface
- **Bootstrap** - Responsive design
